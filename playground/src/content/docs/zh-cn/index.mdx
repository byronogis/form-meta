## form-meta 是什么？

用类型安全的 raw meta 描述嵌套表单，解析后得到带 `name`/`path`/`fullName`/`parent` 的字段表，直接用于表单 UI 与校验。

<CardGrid stagger>
	<Card title="Flatten & purge" icon="tune">
		按需展开或裁剪对象/数组嵌套，拿到需要的字段形态。
	</Card>
	<Card title="Full name helpers" icon="code-block">
		自动生成带索引的路径，便于注册数组字段（如 sections[2].paragraphs[3].text）。
	</Card>
	<Card title="Schema friendly" icon="checkmark">
		支持 Standard Schema v1（Zod/Valibot/Yup 等）直接用作校验。
	</Card>
	<Card title="Playground" icon="rocket">
		在示例页面用 TanStack Form + Vue 动态渲染解析后的字段。
	</Card>
</CardGrid>
